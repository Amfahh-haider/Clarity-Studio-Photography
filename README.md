📸 Clarity Studio Photography – Pricing Calculator

A modern, interactive photography pricing calculator that allows users to build a custom quote by selecting session duration, editing options, and album packages.

The interface features a luxury aesthetic UI with glassmorphism, gradients, and smooth interactions, making it ideal for photography studios or creative service businesses.

✨ Features

Interactive Pricing Builder

Adjust session duration with a slider

Add or remove photo edits

Select album packages (None, Basic, Premium, Deluxe)

Real-Time Cost Calculation

Automatic price updates

Detailed estimate breakdown

Dynamic total price display

Professional UI Design

Dark luxury theme

Glassmorphism-style cards

Smooth animations and hover effects

Elegant typography using Google Fonts

Quick Quote Sharing

One-click button copies quote details to clipboard

Perfect for sending estimates to clients
